What is CSS?

CSS is the language we use to style an HTML document.
CSS describes how HTML elements should be displayed.
CSS is used to define styles for your web pages, including the design, layout and variations in display for different devices and screen sizes.

    This is a paragraph, shown in the Times New Roman font. You can get it if you create class for paragraph style. For examle:

    " .p1 { font-family: "Times New Roman", Times, serif;}"


    This is a paragraph, shown in the Arial font and red color. You can get it if you create class for paragraph style. For examle:

    ".p2 { font-family: Arial, Helvetica, sans-serif;color:red;}"


    This is a paragraph, shown in the Lucida Console font ,color, bold and italic style. You can get it if you create class for paragraph style. For examle:

    ".p3 { font-family: "Lucida Console"; font-weight: bold; font-style:italic; color:rgba(10, 195, 205);}"
